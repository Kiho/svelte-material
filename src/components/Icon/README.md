## Icon

### Markup

```html
<MIcon icon="edit" on:click="clicked()" />
```

### Script 

```javascript
methods: {
    clicked() {
        // do something
    }
}
```

### Props

| Prop | Type | Default | Required | Description |
|------|------|---------|----------|-------------|
| icon | String | - | true | material icon name |

Events are mapped to the inner element.
